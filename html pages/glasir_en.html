Glasir

Model and analyze the security of your systems. Easier. Faster.

What is Glasir ?

Glasir is an open-source software dedicated to the security analysis of your systems. With Glasir, you will be able to identify your system's weak-points using "Attack-Defense Tree".

Glasir is the product of a project conducted by 4th's year IT students at the Insa of Rennes. The initial objective of that project was to localize the weaknesses of the public transports system in Rennes. In order to achieve that, we have decided to create a analyze-tool, based on the Attack-Defense Tree formalism.

What are "ADTrees" ?

"ADTree" is short for "Attack-Defense Tree", a graphical way to represent the possible attacks and/or defenses concerning a given system. ADTree is an improvement of the older "Attack Tree" formal representation, where only attacks could be represent on the Tree. By adding Defense to the representation, ADTrees allow to see dynamically the impact of defenses on the system. ADTrees also allow you to start from the attacker rather than from the system to defend, to be the more accurate given a specific type of attacker.

To start with : ADTool

ADTool is an open-source software developed in order to computerize ADTrees. In addition to their graphical representation, it allows to valuate the ADTrees with several parameters, which are propagate on the entire tree using "domains", a recursive method to evaluate the reachability of an upper node from lower nodes. Glasir uses this tool as an ADTree viewer and editor.

How to analyze ADTrees ?

Glasir contains several tools for the analysis of systems. Three modules are available for this purpose :

  • the Function Editor, which allows to express compromises between attributes of the attack by creating a new attribute which is function of two different attributes ;
  • the Filter, to highlight the possible attacks given a specific attacker, by selecting only reachable nodes for him depending on his abilities ;
  • the Optimizer, to identify the most effective/less costly attack on the system for a specific attribute or parameter.
